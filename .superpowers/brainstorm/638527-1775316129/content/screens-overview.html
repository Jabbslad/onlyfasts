Section 3: App Screens & Navigation

The app has 5 main screens accessed via a bottom tab bar. Tap to select the structure that feels right.

A

5 Tabs — Full Feature Parity

Timer (home) • Log (meals) • Hydration • Progress (streaks/badges/history) • Settings

Matches Zero's structure closely. Each feature gets its own dedicated screen.

B

4 Tabs — Combined Progress

Timer (home) • Log (meals + hydration combined) • Progress (streaks/badges/history) • Settings

Simpler nav. Meals and hydration are both "things you consume" so they share a screen with tabs.

C

3 Tabs — Dashboard Home

Dashboard (timer + today's meals + hydration) • Progress (streaks/badges/history) • Settings

Most compact. Everything for "today" lives on one scrollable dashboard. Fewer taps for daily use.
